# Tube English
YouTube 로 영어공부하는 데 도움이 되는 기능들을 쉽게 접근할 수 있도록 도와주는 웹 사이트

# 주요 기능
* 구간 반복
* 속도 조절
* 단어 사전
* 자막 on/off
* 5/10초 앞으로 가기

# 개발환경
* javascript
* app-engine with golang
* youtube api (data + player api)

# 개발 사이트
http://tube-english.appspot.com/
